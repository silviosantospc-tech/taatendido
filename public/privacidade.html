Política de Privacidade

Última atualização: 03/05/2026

1. Quais dados coletamos

O TáAtendido pode armazenar dados necessários para operar o atendimento, como nome, email, telefone, empresa, contatos, mensagens, respostas rápidas, produtos, configurações do atendimento e registros de acesso.

2. Para que usamos os dados

Usamos os dados para autenticar usuários, organizar conversas, permitir atendimento pelo WhatsApp, salvar histórico, gerar respostas operacionais, prestar suporte e manter segurança do sistema.

3. Onde os dados ficam armazenados

Os dados ficam armazenados em infraestrutura de banco de dados e aplicação usada pelo TáAtendido. O acesso é restrito conforme perfil de usuário e empresa.

4. Segurança

Aplicamos controles como HTTPS/SSL, login protegido, isolamento por empresa, limitação de tentativas, cookies seguros, auditoria administrativa e criptografia de novos dados sensíveis quando a chave de criptografia está configurada.

5. Compartilhamento

Não vendemos dados pessoais. Dados podem ser processados por serviços necessários à operação, como hospedagem, banco de dados e integrações de mensagens, sempre limitados à finalidade do serviço.

6. Direitos do titular

O titular pode solicitar acesso, correção ou exclusão de dados pessoais, conforme aplicável pela LGPD. Solicitações devem ser enviadas pelo canal de contato informado nesta página.

7. Retenção e exclusão

Os dados são mantidos enquanto forem necessários para operação, obrigações legais, segurança e suporte. O TáAtendido pode aplicar política de retenção automática para dados antigos.
